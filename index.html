My Own Server At Home

This website is hosted in Github at the moment. However, I am planning on self-host it in my own server at home using a Raspberry Pi.

  • The idea of this project is to learn the magic behind self-hosting.
  • How to use the command line in linux to move around my server.
  • As well as deploy containers using Docker and Docker Compose
