Pi Server

My Own Server At Home

This website is hosted in Github at the moment. However, I am planning on self-host it in my own server at home using a Raspberry P 4. For those who does not know, a Raspberry Pi is a single-board computers of the size of a credit-cart. It is a low cost device developed in UK

The Raspberry Pi foundation is a project originally leaned towards the promotion of teaching basic computer science in schools and in developing countries

- more info -


    The idea of this project is:
  • To learn the magic behind self-hosting.
  • To learn how to use the command line in Linux and move around my server as a pro.
  • To learn Git & github.
  • To learn Docker and Docker Compose.
  • To deploy containers and learn how to use Portainer.
  • To build my first Blog. To document the process and be able to go back in time and recreate the project in need it.
  • To Develope frontEnd and BackEnd skills
  • To sufil my needs of having to store private information. a centralize storage unit.
